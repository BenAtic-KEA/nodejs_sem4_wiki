Terminal commands

In this first section you will be able to see the most used commands for different tools

Git

# New repository
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin [URL]
git push -u origin main
# Add all changes 
git add .
# Commit all added changes 
git commit -m "my message"
# Push commit to repository
git push
# Pull changes from repository
git pull
# Help 
git help
NPM

# Version
npm version
# install dependencies
npm i [package name]
# Run Script 
npm run [script name]
# Help 
npm help
NodeJS

# Open NodeJS shell
node
# Run project
node app.js
# Help 
node
.help
Nodemon

# Run project
nodemon app.js
# Help 
nodemon -h

Scripts

Scripts are used for different purposes. Normally its used to secure that anyone who try to run the project will get the right environments or other modification at start.

A common use is to make a "dev" script which is running with "nodemon" and some local environments

This is because nodemon is a dev tool which redeploy your project after every change in specified files

Example of a script:

"scripts": {
    "start-dev": "cross-env PORT=8080 nodemon app.js",
    "start-prod": "node app.js"
    }
