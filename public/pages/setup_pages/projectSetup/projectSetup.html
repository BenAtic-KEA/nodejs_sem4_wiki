NodeJS

NodeJS is a cross-platform runtime system which is used in alot in development of server-side webaplications.

The system is written in Javascript and is usable on a wide choice of operation systems

Npm

Npm(Node Package Manager) is a package manager tool, which by default is used for NodeJS.

To interact with npm we make use of different Terminal commands so we can install different packages/dependencies

Step 1

Create package.json with {} brackets and do following terminal commands:

npm i express
After installing express you have to open the package.json and apply type: "module" as following: 

{
    "type": "module",
    "dependencies": {
        "express": "^4.18.2",
    }
}

→

Step 2

Now you're ready to add app.js in the root of the project.

app.js with express is our way to get requests from a user

So now we import express by using the type "module" and setup following:


import express from "express";
const app = express();
app.use(express.static("public"));
app.use(express.json());

const PORT = process.env.PORT || 8080;

app.listen(PORT, (error)=> {
    if(error){
        console.log(error)
    }
    console.log(`server is running on port`
        ,PORT)
})

  • To secure whats public for the client we make a "public" folder and set this as the location through express.static("public")

  • In the end we define which Port to listen to, when our application is live
