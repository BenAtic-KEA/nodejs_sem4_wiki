NodeJS

NodeJS is a cross-platform runtime system which is used in alot in development of server-side webaplications.

The system is written in Javascript and is usable on a wide choice of operation systems

Npm

Npm(Node Package Manager) is a package manager tool, which by default is used for NodeJS.

To interact with npm we make use of different Terminal commands so we can install different packages/dependencies

Step 1

Create package.json with {} brackets and do following terminal commands:

npm i express
After installing express you have to open the package.json and apply type: "module" as following: 

{
    "type": "module",
    "dependencies": {
        "express": "^4.18.2",
    }
}
