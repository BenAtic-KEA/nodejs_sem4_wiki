NodeJS

NodeJS is a cross-platform runtime system which is used in alot in development of server-side webaplications.

The system is written in Javascript and is usable on a wide choice of operation systems

Npm

Npm(Node Package Manager) is a package manager tool, which by default is used for NodeJS.

To interact with npm we make use of different Terminal commands so we can install different packages/dependencies

Express

Express is a backend web application framework which is used in coperation with NodeJS

It's written in Javascript and is used to create web applications and API's

In our projects its what we used to setup our app.js to listen on a specific port and by that being able to send a response

Nodemon

Nodemon is a dev tool which redeploy your project after every change in specified files

Its used only as a dev tool since you dont want to keep redeploying your live projects after every change in the files.
