Definition

Rest API origins from "RESTful API", and is a architectural way of delivering data.
Its purpose is to deliver data in a streamlined manor so you have a more generalised way of recieving data. A common way is to deliver the response in JSON, XML, HTML etc.. since its a universal language cross platforms.

API

API stands for Application Programming Interface.
It's a contract between two applications which defines the way of communication, through "Request" & "Response"

API's is best illustrated through client - server communication, where the client has to send a "Request" to the server, and the servere respond with a "Response"

Rest

REST stands for Representational State Transfer, and the main point for REST API is to be stateless

REST is defined by following functions:

  • GET
  • POST
  • PUT
  • DELETE

The main point is to send plain data without any form of manipulation.
