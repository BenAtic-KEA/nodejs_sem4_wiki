Functions

Callback functions

A callback function is a function passed as an argument to another function.

This makes it possible for a function to call another function.

Example:

function genericActionExecutor(callbackFunction, name){
    // execute some code
    return callbackFunction(name);
}

const spitting = (name) => `${name} is spitting.`;


console.log(genericActionExecutor(spitting,"Mark"));
// prints: Mark is spitting.
Arrow Functions

A arrow function makes it possible to write functions cleaner than a regular function

It makes it possible to make one liners

Example:

function gActionExecutor(callbackFunction, name){
    // execute some code
    return callbackFunction(name);
}

gActionExecutor((name) => `${name} is running away.`,"Mark");
// this prints out Mark is running away
